![fastboot](fastboot.png)

### fastboot进入方法（不适用于变砖后卡第一屏的情况）
- 方法①：连接手机，输入adb命令：adb reboot bootloader

- 方法②：关机并等待15秒，长按电源键和音量-，出现震动后松手
